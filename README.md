# Follow-up Reminder System

Automated system to track leads and send reminders for follow-ups.

---

## Overview

This system ensures no lead is forgotten.

It:

* Tracks last contact date
* Identifies inactive leads
* Sends reminder notifications

---

## Features

* Lead tracking
* Follow-up reminders
* Google Sheets integration
* Simple CRM logic

---

## Data Structure

Name | Email | Last Contacted | Status

---

## Setup

### Step 1

Create Google Sheet named:
Leads

---

### Step 2

Add columns:
Name | Email | Last Contacted | Status

---

### Step 3

Open Apps Script

---

### Step 4

Paste script from:
/scripts/reminder.js

---

### Step 5

Run:
checkFollowUps()
